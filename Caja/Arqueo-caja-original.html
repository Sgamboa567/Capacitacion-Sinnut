🧾 Arqueo de caja en el ERP

En este módulo aprenderás cómo realizar el arqueo de caja en Sinnut. Sigue los pasos y utiliza los recursos para dominar el proceso.

⏱ Tiempo estimado de operación: 2:00 minutos

🎬 Video: Arqueo de caja

📋 Pasos para realizar el arqueo de caja

Paso 1: Ingresa al módulo Gestión de contabilidad sub modulo Arqueo de caja.
Paso 2: Selecciona el icono + para un nuevo arqueo de caja.
Paso 3: Registra los valores de las cajas.
Paso 4: Confirma el saldo final y genera el reporte de arqueo.
